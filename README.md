## Tech Stack
### **Backend**
- **Framework:** Flask (Python)
- **Database:** PostgreSQL
- **Authentication:** Flask-JWT-Extended
- **API Design:** Flask-RESTful
- **Migrations:** Flask-Migrate

### **Frontend**
- **Library:** ReactJS (Redux Toolkit)
- **State Management:** Redux Toolkit
